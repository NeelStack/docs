# NeelStack Docs Repository

Welcome to the official engineering documentation and standards repository of **NeelStack**.

## Repository Structure

```text
docs/
│
├── README.md                                  # Repository Overview
├── CONTRIBUTING.md                            # Contribution Guidelines
├── CHANGELOG.md                               # Documentation Changelog
├── LICENSE.md                                 # Repository License
│
├── engineering-system/                        # ⭐ Engineering Constitution
│   ├── 00-foundation/                         # NES Foundation standards
│   ├── 01-architecture/                       # NES Architecture standards & principles
│   ├── 02-engineering-laws/                   # Engineering Laws
│   ├── 03-technology/                         # Approved technology stacks
│   ├── 04-repository/                         # Repository standards & governance
│   └── 05-21/                                 # Additional Engineering Phases (Backend, Frontend, AI, etc.)
│
├── product-docs/                              # Product specific documentation
│   ├── eduos/
│   ├── toolvines/
│   ├── dhruvaos/
│   ├── naukarimitra/

│
├── api/                                       # API Design & specs
├── architecture/                              # High level architecture designs
├── playbooks/                                 # Runbooks and operations playbooks
├── onboarding/                                # Onboarding guides
├── standards/                                 # General standards
└── archive/                                   # Archived resources
```

## Documentation Hierarchy

```
Vision → Mission → Engineering Philosophy → Engineering Laws → Architecture Principles → Engineering Standards → Templates → Reference Implementations → Products
```

*Nothing below can violate anything above.*

## Numbering Convention

| Prefix | Meaning |
|---|---|
| **NES** | NeelStack Engineering Standard |
| **NAP** | NeelStack Architecture Principle |
| **LAW** | Engineering Law |
| **ADR** | Architecture Decision Record |
| **RFC** | Request for Comments |
| **NST** | Template |
| **NSC** | Checklist |

## Engineering Maturity

| Level | Meaning |
|---|---|
| **L0** | Draft |
| **L1** | Working |
| **L2** | Production |
| **L3** | Enterprise |
| **L4** | Platform |
| **L5** | Gold Standard |

Every document in this repository must eventually reach **L5**.
