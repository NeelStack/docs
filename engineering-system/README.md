# NeelStack Engineering Constitution

This directory contains the engineering standards, architecture principles, laws, and templates for NeelStack.

## Directory Structure

- `00-foundation/`: Core company mission, vision, engineering culture, and maturity standards.
- `01-architecture/`: System design, architecture principles, and framework compliance.
- `02-engineering-laws/`: Binding engineering laws that govern code quality, domain ownership, and security.
- `03-technology/`: Technology stack selections, database, backend, frontend, devops, and AI platform guidelines.
- `04-repository/`: Monorepo, directory structure, package rules, and governance.
- `05-21/`: Placeholders for specialized engineering domains (e.g. backend, frontend, devops, platform, security).
- `appendix/`: Acronyms, reference docs, index, and bibliography.
- `assets/`: Shared diagrams, logos, icons, and layout templates.
